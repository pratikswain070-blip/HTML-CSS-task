 Where does gap add space?
Between all boxes - left, right, up, down. Creates empty space inside the grid.

gap vs margin - what's different?
gap = space BETWEEN boxes
margin = space OUTSIDE boxes

Does gap make container bigger?
NO container stays same size. Gap takes some space from inside.

Can we use gap for only rows/columns?
YES! Use row-gap for rows only, column-gap for columns only, or gap for both.

Why use gap in Grid?
Super simple One property = done. Margin = complicated.
