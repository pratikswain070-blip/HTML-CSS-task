Which layout breaks faster on small screens?
The px layout breaks faster. If columns are 200px each and screen is only 300px wide, nothing fits properly.

Which layout adjusts automatically?
The fr layout adjusts automatically because it divides available space.

Why is fixed layout risky?
Fixed layouts don't respond to different screen sizes. 

Where can fixed columns be useful?
It can be in sidebars or very specific areas where we want things to stay the same size. 

Which unit would you choose for a responsive website?
I would choose fr because it works on all screen sizes.
